The Software Defined Storage devroom will go into it's seventh round for talks around Open Source Software Defined Storage projects, management tools and real world deployments.

Presentation topics could include but are not limited too:

  • Your work on a SDS project like Ceph, Gluster, OpenEBS, CORTX or Longhorn
  • Your work on or with SDS related projects like OpenStack SWIFT or Container Storage Interface
  • Management tools for SDS deployments
  • Monitoring tools for SDS clusters
