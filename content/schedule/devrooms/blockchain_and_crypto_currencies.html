The Distributed Web devroom is a place for folks to learn about and play with the P2P, distributed, secure internet is emerging, distributed, peer-to-peer, secure internet. Sessions will include demos, presentations, panels, Q&As, hands-on learning opportunities, facilitated experiences, networking spaces, on-boarding sessions, and discussions of connection points between frameworks/protocols. Sessions will be led by members of each project’s core team and community. The Distributed Web dev room content supports the DWeb values: decentralization, data sovereignty, security, privacy, and censorship-resistance.  This year presentations will range from new and emerging post-blockchain technologies, to hardware wallets to exciting new project using this technology in innnovative ways.
